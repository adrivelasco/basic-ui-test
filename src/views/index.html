CODE CHALLENGE

Hola! Antes de empezar, te recomendaría dedicarle un tiempo a entender la estructura del proyecto.

  • /mock: Contiene dos archivos de formato JSON con la información que posteriormente requerirás para realizar el challenge.

  • /node_modules: Contiene todas las dependencias necesarias para levantar y trabajar sobre el proyecto.

  • /public: Es el output que genera Webpack a partir del código source.

  • /src: Te enfocarás principalmente en este directorio debido a que es el punto entrada y el código base.

EJERCICIOS

1. 404 NOT FOUND! - DIFICULTAD: MEDIA

Cuando intentamos ingresar a una url inválida (Por ej: "/inexistente") se muestra una vista indicando que hay un 404 ERROR pero para los responsables de Marketing este mensaje no es para nada user-friendly. Por lo cual, crea la tarea para el equipo de Front-End de: Repetir el menú de navegación de la Home; mostrar un mensaje explicándole al usuario de lo que está ocurriendo; y un botón con el texto "Volver a Inicio" que redireccione hacía la Home al presionar sobre él.
